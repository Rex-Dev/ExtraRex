# [Extra Rex TDBot v1.0](https://telegram.me/RexCompany)

*An advanced and powerful administration bot based on telegram-bot*

## Installation


```sh

# Let's install the Cli bot.


cd $HOME
git clone https://github.com/Rex-Dev/ExtraRex.git

cd Rexcli
chmod +x rex.sh
./rex.sh install
./rex.sh config

# Login Cli Account
./rex.sh logcli
# Login Api Bot
./rex.sh logapi

# For Autolaunch

cd Rexcli
chmod 777 autorex.sh
screen ./autorex.sh

# Delete Config & Login Data
./beyond.sh reset
```
```sh

# Let's install the Api bot.

cd $HOME
cd Rexapi
chmod +x rex.sh
./rex.sh install
./rex.sh config

./rex.sh

# For Autolaunch

cd Rexapi
chmod 777 autorex.sh
tmux
./autorex.sh
```
